Documento HTML


Sumário


O elemento HTML <html> (ou HTML Root element ) epresenta a raiz de um HTML ou XHTML documento. Todos os outros elementos devem ser descendentes desse elemento.


  • Categorias de conteúdo nenhuma
  • Conteúdo permitido Um elemento <head>, seguido por um elemento <body>.
  • Omissão de tag A tag inicial pode ser omitida se a primeira coisa dentro do elemento <html>  não for um comentário.
    a tag final pode ser omitida se o elemento <html>   não for imediatamente seguido por um comentário, e conter um elemento <body> ou que não está vazio, ou cuja tag inicial está presente.
  • Elementos pais permitidos Como o elemento raiz do documento, ou qualquer fragmento de subdocumento é permitido em um documento composto.
  • DOOM interface HTMLhtmlElement (en_US) (Disponível apenas em inglês.)

Atributos


Esse elemento inclui os atributos globais

mainfest
Especifica a URI de um recurso manifest indicando os recursos que devem ser carregados na memória local.

version
Descontinuado desde o HTML4.01 Obsoleto desde o HTML5
Especifica a versão da definição de tipo de documento HTML que governa o documento atual. Esse atributo não é necessário, porque é redundante com a informação de versão na declaração de tipo de documento.
