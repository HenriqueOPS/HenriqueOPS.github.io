Referência HTML


Esta referência de HTML descreve todos os elementos e atributos do HTML,
incluindo Atributos globais que se aplicam em todos os elementos.


Referência de elementos do HTML
Esta página lista todos os elementos do HTML que são criados usando tags.

Exemplos

  • <base>
    • O elemento HTML Base (<base>) especifica o endereço (URL) utilizada por
    todos os endereços relativos contidos dentro de um documento.
    Há um número máximo de 1 (um) elemento Base <base> do documento.
  • <head>
    • O elemento HTML <head> providencia informações gerais (metadados) sobre
    o documento, incluindo seu título e links para scripts e folhas de estilos.
  • <link>
    • O Elemento HTML <link> especifica as relações entre o documento atual e
    um recurso externo. Possíveis usos para este elemento incluem a definição de
    uma estrutura relacional para navegação. Este elemento é mais usado para
    vincular as folhas de estilo.

Referência de atributos do HTML
Os elementos do HTML contêm atributos, que são valores adicionais que configuram os
elementos ou ajustam seu comportamento de várias formas, para que se adaptem aos
critérios dos usuários.

Exemplos

  • class
    • Frequentemente usado com CSS para estilizar os elementos com propriedades comuns.
  • href
    • A URL de um recurso ligado.
  • id
    • Frequentemente usado com CSS para estilizar um elemento específico. O valor deste atributo deve ser único.

Atributos globais
Atributos globais são atributos comuns a todos os elementos do HTML;
podem ser usados em todos os elementos, embora possam não ter efeito
em alguns elementos.

Exemplos

  • accesskey
    • Fornece uma dica para gerar um atalho de teclado para o elemento atual.
    Este atributo consiste em uma lista de caracteres separados por espaços.
    O navegador deve usar o primeiro que existe no layout do teclado do computador.
  • hidden
    • Um atributo booleano indica que o elemento ainda não é relevante ou não é mais relevante.
    Por exemplo, ele pode ser usado para ocultar elementos da página que não podem ser usados
    até que o processo de login tenha sido concluído. O navegador não renderizará esses elementos.
    Este atributo não deve ser usado para ocultar conteúdo que possa ser legível.
  • class
    • É uma lista separada por espaço das classes do elemento. as classes permitem ao CSS e ao JavaScript
    selecionar e acessar elementos específicos atraves dos seletores de classe ou funciona como um método
    Document.getElementsByClassName().

Tipos de links
Em HTML, os tipos de links a seguir indicam o relacionamento entre dois documentos,
em que um aponta para o outro usando um elemento <a> , <area> ou <link> .

Exemplos

  • external
    • Indica que o hiperlink leva a um recurso fora do site da página atual; ou seja, seguir o link fará com que o usuário saia do site.
  • nofollow
    • Indica que o documento vinculado não é endossado pelo autor deste, por exemplo, se não tiver controle sobre ele,
    se for um mau exemplo ou se houver relação comercial entre os dois (link vendido). Esse tipo de link pode ser usado
    por alguns mecanismos de pesquisa que usam técnicas de classificação de popularidade.
  • tag
    • Indica que o hiperlink se refere a um documento que descreve uma marca que se aplica a este documento.
    Observação: esse tipo de link não deve ser definido em links para um membro de uma nuvem de tags, pois
    eles não se aplicam a um único documento, mas a um conjunto de páginas

Categorias de conteúdo
Todo elemento em HTML é membro de uma ou mais categorias de conteúdo
— estas categorias agrupam elementos que compartilham características comuns.
